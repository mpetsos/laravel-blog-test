API Documentation: Laravel Blog test

Base URL: http://localhost:8000

Endpoints Index

  • GET /posts - Retrieves a list of all available posts.
  • POST /posts - Create a new post.
  • GET /posts/{$postId}/{$slug} - Get a post by id and slug.
  • PUT /posts/{$postId} - Update a post by id (if is the author).
  • DELETE /posts/{$postId} - Delete a post by id (if is the author).
  • POST /posts/{$postId}/comments - Create a new comment for post by id.
  • GET /users/{$userId}/posts - List posts by user id.
  • GET /users/{$userId}/comments - List comments by user id.
  • GET /categories - List all categories.
